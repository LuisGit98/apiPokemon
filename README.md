# Ejecutar en desarrollo 
1. clonar repo
2. Ejecutar 
```
npm i 
```
3. tener Nest cli instalado
```
npm i nest cli install
```
4. levantar la db
```
docker-compose up -d
```
## Stack usado 
* mongo 
* Nest
